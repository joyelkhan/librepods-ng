# LibrePods NG User Manual

## Installation

### Linux (AppImage)
```bash
chmod +x LibrePods-x86_64.AppImage
./LibrePods-x86_64.AppImage
```

### Windows (MSIX)
Double-click `LibrePods-1.0.0.msix` to install via Microsoft Store.

### macOS (DMG)
1. Open `LibrePods-1.0.0.dmg`
2. Drag LibrePods to Applications folder
3. Launch from Applications

### Android (APK)
Install via Magisk module or direct APK installation.

## Getting Started

1. **Scan for Devices**: Tap "Scan" to find nearby AirPods
2. **Connect**: Select your AirPods from the list
3. **Configure**: Adjust ANC mode, transparency, and other settings
4. **Monitor**: View battery status and device information

## Features

### ANC Control
- Off
- Active Noise Cancellation
- Transparency Mode
- Adaptive Mode

### Battery Monitoring
Real-time battery percentage for both earbuds and case.

### Device Information
- Firmware version
- Model identification
- Feature availability

### Advanced Settings
- Custom transparency levels
- Head gesture detection
- Hearing aid mode
- Spatial audio configuration

## Troubleshooting

### Device Not Found
- Ensure Bluetooth is enabled
- Restart the app
- Reset AirPods (hold setup button 15 seconds)

### Connection Drops
- Move closer to device
- Reduce interference from other Bluetooth devices
- Update firmware if available

### Battery Not Updating
- Disconnect and reconnect
- Force close and reopen app
- Check Bluetooth permissions

## Privacy & Security

LibrePods NG uses AES-256-GCM encryption for all communications. No data is sent to external servers.
